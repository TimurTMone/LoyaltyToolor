# TOOLOR — Статус разработки

**Версия:** 1.0.0 (билд 11)
**Дата:** 4 апреля 2026
**Платформа:** iOS (TestFlight)

## Работает
- Каталог на 111 товаров с сайта toolorkg.com (фото, цены, размеры, цвета)
- Авторизация по SMS-коду, профиль, онбординг
- Корзина, оформление заказа, самовывоз/доставка
- Программа лояльности: 4 уровня, кэшбэк 3–12%, QR-код для сканирования в магазине
- Оплата через Finik (карта/QR)
- Реферальная программа, промокоды, списание баллов при покупке
- Админ-панель: дашборд, управление товарами/заказами/пользователями, сканер QR, аналитика

## Не работает / в процессе
- Push-уведомления (UI готов, Firebase не подключён — пока polling)
- SMS-провайдер для OTP (сейчас код приходит в ответе, нужен реальный SMS-шлюз)
- AI-чат (работает базовая версия, полная на Claude — нужен API-ключ)

## На этой неделе
- Подключение реального SMS-провайдера
- Firebase push-уведомления
- Финальная настройка Finik webhook (нужен секретный ключ от Finik)

## Безопасность
Обновили rate limiting на auth/chat, HMAC-подпись для webhook, вынесли ключи Finik из исходников.
